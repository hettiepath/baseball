# 분석 방법
1) 문제 정의 및 분석 목표 세우기
- 둘로 나누어 예측
- 머신러닝 모델 활용 -> 정확도 높은 모델 제안 -> 중요한 변수 도출

2) 데이터 전처리
- 결측치 처리
- 컬럼 추가 / 정리

3) 시각화
- 변수별 관계 시각화 -> 비율 도출

4) 통계 분석
```
1. 상관 분석
2. T-test : 컬럼 제외
3. 카이제곱 검정 : 비율 차이 비교
4. PCA 주성분 분석 (screeplot 시각화)
```

5) 머신러닝 적용
- 랜덤포레스트, 의사결정나무(decision tree) -> 변수 중요도를 도표로 시각화

6) 결론
```
1. 가장 정확도 높은 모델의 비율
2. 중요 변수 도출
```

7) 요구사항
- 엑셀, R : 비율 시각화
- SAS : 통계 분석
- python : 머신러닝 적용(colab)
